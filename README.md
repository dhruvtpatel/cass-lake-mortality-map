# Cass Lake mortality map

Interactive map of Minnesota mortality, 2011–2024, centered on the former
St. Regis / Cass Lake Superfund site. Zoom/pan enabled; population-adjusted.

**Metrics (toggle):**
- Death rate (vs neighbors) — age-adjusted, auto-zoomed to the Cass Lake region
- All-cause death rate — per capita, by ZIP-code area
- Dioxin-related death rate — per capita (Cass Lake: 86th percentile statewide)
- Youngest-dying (age at death) — median age at death (Cass Lake: 63, younger than 98.7% of MN communities)

**Overlay layers:** federally recognized reservation boundaries, Minnesota
Superfund (NPL) sites, 15/25/50 km distance rings. Light/dark mode.

**Data:** Minnesota death-certificate records (2011–2024), aggregated to ZIP-code
areas (no individual-level data). Population denominators from U.S. Census / ACS.
Boundaries from Census TIGER; Superfund sites from EPA. Built with D3.
