# Cass Lake mortality map

An interactive map of Minnesota death rates by ZIP-code area, 2011–2024,
highlighting the community around the former St. Regis / Cass Lake Superfund site.

Views (all population-adjusted):
- **Death rate (vs neighbors)** — age-adjusted, Cass Lake vs surrounding towns
- **All-cause death rate** — per capita, by ZIP area
- **Dioxin-related death rate** — per capita (Cass Lake: 86th percentile statewide)
- **Youngest-dying** — median age at death (Cass Lake: younger than 98.7% of MN communities)

**Data:** Minnesota death-certificate records (2011–2024), aggregated to ZIP-code
areas (no individual-level data). Population denominators from U.S. Census / ACS.
ZIP-area boundary from Census TIGER. Rates are age-adjusted where noted; statewide
per-capita views are crude. Built with D3.
